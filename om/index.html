Om data1.no

data1.no er et gratis, norsk verktøy for å analysere e-postsikkerheten til ethvert domene. Verktøyet er laget av Micronet AS og er åpent for alle — ingen registrering, ingen betaling.

Hvorfor vi laget dette

Som IT-konsulenter i Micronet AS møter vi jevnlig bedrifter som har blitt utsatt for direktørsvindel eller phishing. I de fleste tilfellene er det teknisk mulig å forhindre dette med riktig konfigurasjon av SPF, DKIM og DMARC.

Eksisterende verktøy er enten på engelsk, krever betalt abonnement, eller gir tekniske svar uten kontekst som norske IT-ansvarlige kan handle på. data1.no er svaret vi ville hatt tilgang til selv.

67 % av norske bedriftsdomener mangler DMARC med p=reject — den eneste policyen som faktisk blokkerer e-post-forfalskning. data1.no hjelper bedrifter å forstå hvor de står og hva de må gjøre.

Hva vi analyserer

SPF
Sender Policy Framework — hvem kan sende e-post i ditt navn?
DMARC
Domain-based Message Auth — hva skjer med falsk e-post?
DKIM
DomainKeys Identified Mail — er e-posten kryptografisk signert?
MTA-STS
Mail Transfer Agent Strict Transport Security — er transporten kryptert?
TLS-RPT
TLS Reporting — mottar du rapporter om krypteringsfeil?
BIMI
Brand Indicators for Message Identification — vises logoen din i innbokser?

Personvern og data

data1.no setter ingen cookies, sporer ikke individuelle brukere og lagrer ikke søkehistorikk knyttet til deg. Vi loggfører kun anonymiserte aggregerte besøkstall — uten samtykkekrav, i tråd med GDPR.

Domenene du søker på analyseres i sanntid mot offentlige DNS-poster og lagres ikke.
