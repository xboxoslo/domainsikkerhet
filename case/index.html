Hvorfor case studies?

Vi tror på åpenhet — å vise hva som faktisk fungerer i praksis er bedre enn løse løfter. Hver case viser:

  • Utgangspunkt — karakter på data1.no før tiltak (typisk F eller D)
  • Konkrete tiltak — nøyaktig hvilke DNS-endringer og hvordan
  • Tidsbruk — hvor lenge fra start til full p=reject
  • Resultat — sluttkarakter (typisk A eller A+) og ev. observerte effekter
